# 🚀 Техническое задание

## Проект: Premium Telegram Bot Development Studio (3D Concept)

------------------------------------------------------------------------

# 1. 📌 Общая информация

**Цель проекта:**\
Разработка премиального корпоративного сайта для студии разработки
Telegram-ботов с современным 3D-концептом (three.js стиль),
ориентированного на конверсию и имидж технологичного бренда.

**Формат сайта:**\
Многостраничный сайт с лендинг-главной страницей.

**Целевая аудитория:**\
- Владельцы бизнеса\
- Digital-агентства\
- Стартапы\
- E-commerce проекты\
- SaaS компании

------------------------------------------------------------------------

# 2. 🧱 Технологический стек

## Frontend

-   TypeScript
-   React
-   Next.js (App Router)
-   Tailwind CSS
-   Shadcn UI
-   Radix UI
-   Framer Motion
-   React Three Fiber (three.js)

## Backend

-   Nest.js (TypeScript)
-   PostgreSQL
-   Prisma ORM
-   REST API
-   JWT авторизация (для админки)

## DevOps

-   Frontend: Vercel
-   Backend: Railway / Render
-   Docker (опционально)
-   CI/CD (GitHub Actions)
