# 3D Animation System Documentation

Эта документация описывает систему 3D-анимаций, разработанную для Gerusta. Система построена на базе **Framer Motion** и **Tailwind CSS**, обеспечивая высокую производительность (60fps) и простоту использования.

## 1. Компоненты

### `TiltCard`
Создает интерактивный 3D-эффект наклона при наведении курсора.

**Параметры:**
- `maxTilt` (number): Максимальный угол наклона в градусах. По умолчанию: `15`.
- `scale` (number): Масштаб при наведении. По умолчанию: `1.05`.
- `perspective` (number): Глубина перспективы. По умолчанию: `1000`.

**Пример:**
```tsx
<TiltCard maxTilt={10}>
  <div style={{ transform: 'translateZ(20px)' }}>Я парю над картой!</div>
</TiltCard>
```

---

### `ParallaxLayer`
Реализует многослойный параллакс-эффект, зависящий от прокрутки страницы.

**Параметры:**
- `offset` (number): Базовое смещение в пикселях.
- `speed` (number): Коэффициент скорости (0-1). По умолчанию: `0.5`.

**Пример:**
```tsx
<ParallaxLayer speed={0.2} offset={50}>
  <img src="/cloud.png" alt="Cloud" />
</ParallaxLayer>
```

---

### `PerspectiveSection`
Применяет эффект трансформации перспективы ко всей секции при скролле.

**Параметры:**
- `rotationIntensity` (number): Сила вращения. По умолчанию: `10`.
- `scaleIntensity` (number): Минимальный масштаб при удалении. По умолчанию: `0.95`.

**Пример:**
```tsx
<PerspectiveSection rotationIntensity={5}>
  <MyComponent />
</PerspectiveSection>
```

## 2. Оптимизация производительности

Для обеспечения плавности 60fps используются следующие техники:
1. **Hardware Acceleration**: Все компоненты используют CSS-класс `.will-change-transform`, который сообщает браузеру о необходимости выноса элемента на отдельный композитный слой.
2. **Backface Visibility**: Свойство `backface-visibility: hidden` предотвращает мерцание при 3D-трансформациях.
3. **Spring Physics**: Вместо линейных анимаций используются пружины (`useSpring`), что делает движение более естественным и снижает нагрузку на CPU.

## 3. Кросс-браузерная совместимость

- Используется стандартный синтаксис CSS 3D Transforms.
- Для браузеров без поддержки 3D (или с отключенным JS) Framer Motion автоматически применяет безопасные 2D-фоллбеки.
- Мобильная адаптивность: Эффекты наклона (`TiltCard`) автоматически отключаются на устройствах без мыши, заменяясь на легкое масштабирование.
